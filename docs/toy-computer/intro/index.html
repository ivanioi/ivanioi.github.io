Intro

这个项目要干什么?​

我感到非常无聊, 加之生活的反复无常, 让我们在计算机中寻找那一丝不变的永恒和宁静吧.

所以, 我的目标是在学习过程中, 根据课程, 教程, 一切资源搭建计算机关键组件, 比如计算机本身, 操作系统, 网络堆栈, 编程语言, 编译器, 数据库, GUI Desktop......

如何实现?​

我为什么认为这个项目是可以实现的？一个 CRUD 仔如何实现这一切?

一方面, 是因为我的目标是学习, 对最终的产物并没有太高的标准, 我只想走一遍我本该在大学就走完的路, 这些产物本就是每个课程对应的实验课产物.

另一方面, 我相信计算机的简单性, 所以整个过程我将通过First Principle 来去学习它, 并通过抽象来管理整体的复杂性. 但是计算机和理论上的图灵机还是有完全不同的性质，图灵机相关理论可以通过数学公式来推理出来，但是计算机则是具体的硬件，我们无法通过公式来推理出, 它的目标是满足图灵机的要求，所以在学习过程中更多的是解构一台计算机，而非推理出来, 因为计算机是无法推理出来的!
