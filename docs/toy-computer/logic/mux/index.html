Mux

What is mux?​

Type: Combinational Logic Circuits​

Input: n Bit Data Input, log2(n) bit Select Input​

Output: 1 Bit​

function: 根据输入 S 选择输出 N Bit 输入中对应的某一个值.​

Boolean Expression​

Y = !S1!S2D0 + !S1S2D1 + S1!S2D2 + S1S2D3
