Decoder

What is decoder?​

Type: Combinational Logic Circuits​

Input: n Bit​

Output: 2^n Bit​

function: 将 n bit 无符号二进制整数解析为一个 2^n bit 长的唯一标识.​

Boolean Expression​

Y0 = !A!B

Y1 = !AB

Y2 = A!B

Y3 = AB
