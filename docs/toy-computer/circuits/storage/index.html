Storage Circuit

我们知道电容可以储备电子，那么在电路中又该如何储备 bit 呢？

R-S Latch​

R-S Latch 由两个相联的 NAND Gate 组成，它的特性如下:

  1. 只能存储 1 bit 信息，该信息从 a 出输出.
  2. 不能同时将 S, R 设置为 0，否则存储的 bit 处于不确定状态.
  3. 当为 S 端输入 0 时，a 输出 1，我们称之为 setting.
  4. 当为 R 端输入 1 时，a 输出 0，我们成指为 resetting.

该电路非常简单，其真值表如下:

Input SInput Ra(store value)
00invalid
011
100
11previous state

当开启电路时，我们一般并不关注其默认值是什么，主需要关注 setting and resetting 即可！

Gated D Latch​

Gate D Latch 由两个 NAND GATE 和一个 R-S Latch 组成。其属于 R-S Latch 的加强版本，便于时用。其特性如下:

  1. D 输入想要存储的值, WE 即 Write Enable 开关.

该电路更加简单，其真值表如下:

Input DInput WEStore Value
00previous state
10previous state
010
111

Flip-Flop​

该存储单元主要用于带有 Clock Signal 组件的时序电路中. 由于在 ONE Clock Cycle 中，电压状态会产生变化, 而我们想还要存储的值在 ONE Clock Cycle 开始时刻设置，其余时刻保持不变.

如果我们直接把 Clock Signal 连接到 Gate D Lacth 上，那么其值有可能产生波动， 因为有一半的时间 WE 一直处与开启状态。

而 Flip-Flop 解决了这个问题，其只有 ONE Clock Cycle 开始时刻 D 输入才有效!

Memory​

Memory 是由一大堆 Storage Elements, Some Gate 组成的 Addressability 可寻址存储矩阵.

  1. Addressability: 可寻址指我们可以通过二进制数字标识出存储矩阵中的任何一个最小存储单元. 最小存储单元一般都是 byte.
  2. Address Space: 地址空间指存储矩阵中所有唯一地址标识的个数。比如 8GB Memroy 对应的 Address Space 就是 2^32, 其中任何一个地址标识都需要用 32bit 来表示。 我们也称 32 bit 为该计算机的 Word Size.
